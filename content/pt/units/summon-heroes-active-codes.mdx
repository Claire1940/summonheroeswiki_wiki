## Solução de Problemas Comuns com Códigos

Se você descobrir que seus **códigos ativos de summon heroes** não estão funcionando, existem alguns motivos comuns para isso acontecer. Primeiro, verifique se há espaços extras. Copiar e colar pode, às vezes, incluir um espaço no final do código, que o jogo lerá como um caractere incorreto.

Segundo, verifique a idade do servidor. Em alguns jogos do Roblox, novos códigos só funcionam em "Servidores Novos". Se você estiver em um servidor que está rodando há várias horas, tente entrar em uma instância diferente para ver se o código é ativado lá. Por fim, lembre-se de que a maioria dos códigos tem uma data de validade. Se um código tem vários meses, provavelmente foi desativado pelos desenvolvedores para dar lugar a novas recompensas.

| Problema | Solução Potencial |
| :--- | :--- |
| **Código Inválido** | Verifique novamente a ortografia e as letras maiúsculas. |
| **Já Resgatado** | Você só pode usar cada código uma vez por conta. |
| **Código Expirado** | O código não está mais ativo; aguarde por uma nova atualização. |
| **Erro de Conexão** | Verifique sua internet; as recompensas exigem uma sincronização estável. |
