## 召喚システムとユニットのレア度の理解

最強の**サモンヒーローズのヒーローアビリティ**を入手するには、まず中央ハブにある召喚バナー（ガチャ）を利用する必要があります。これらのバナーは動的で、30分ごとにローテーションして異なるユニットセットを提供します。各バナーパックは特定のレア度配分になるよう構成されており、プレイヤーが部隊をアップグレードするための一貫した道筋が確保されています。

| ユニットのレア度 | バナーあたりの数 | 召喚確率 |
| :--- | :--- | :--- |
| **Mythic（ミシック）** | 1 | 0.1% |
| **Legendary（レジェンダリー）** | 1 | 100回召喚で天井（確定） |
| **Epic（エピック）** | 2 | 中 |
| **Rare（レア）** | 3 | 高 |

召喚コストは、1回で100ジェム、10連で1,000ジェムです。100回以内の召喚で上位ティアユニットを保証する「レジェンダリー天井システム」は、**異なるバナー間では共有されない**ことに注意が必要です。「フレームレート」や「テクノマスター」などの特定のレジェンダリーユニットを狙う場合は、天井機能を効果的に発動させるために、単一のバナーセッションにジェムを集中させるようにしましょう。

> **⚠️ 警告：** 序盤はレアやエピックのユニットに「自動売却（Auto Sell）」機能を使用しないでください。低ティアのユニットであっても、星ランクとパワーを上げるための主要な手段である合成プロセスには不可欠です。

## 主要なサモンヒーローズのヒーローアビリティの分析

ゲーム内のすべてのユニットは、その攻撃スタイルと必殺技によって定義されます。チームを編成する際は、大量の「雑魚敵」を一掃できるユニットと、高体力のボスを削りきれるユニットのバランスをとる必要があります。

### 単体攻撃スペシャリスト
レジェンダリーの**フレームレート（Frame Rate）**のようなユニットは、精度とスピードに特化しています。フレームレートの**サモンヒーローズのヒーローアビリティ**は、速射性の高い物理攻撃に焦点を当てています。彼の必殺技は、高速ダッシュのシーケンスで単一のターゲットにパンチの連打を浴びせ、画面上に「数字の壁」を作り出します。これにより、サンセットシティのホテルロビーでのボス戦や、サイバーDJニッキ戦において欠かせない存在となっています。

### 範囲攻撃（AoE）コントローラー
対照的に、**テクノマスター（Techno Master）**や序盤の**マーメイド（Mermaid）**などのユニットは、クラウドコントロール（集団制御）に長けています。テクノマスターは広範囲をカバーするデジタルテーマのエネルギーバーストを利用し、「ナイトメア・サーカス」のウェーブモードにおいて最高の選択肢となります。マーメイドはレア度が低いものの、一貫した範囲ダメージを提供し、序盤のストーリーステージを素早くクリアする必要がある新規プレイヤーにとって極めて重要です。

## 合成システム：ヒーローパワーの強化

2026年のメタにおけるユニークな側面は、柔軟な合成（フュージョン）システムです。ランクアップに同一の重複ユニットを必要とする他のヒーロー収集ゲームとは異なり、このゲームでは同じ星のレア度であればどのユニットでも素材（エサ）として使用できます。これにより、特定のユニットを戦闘で使用するかどうかにかかわらず、すべての召喚に価値が生まれます。

### 効果的なユニット合成方法
1.  **メインユニットの決定：** 上位ティアのユニット（例：星4のドラグーン）を選択します。
2.  **素材の収集：** 同じ星レア度の他のユニットを集めます（素材はメインユニットと一致している必要はありません）。
3.  **フュージョンクリスタルの使用：** アイテムショップで購入するか、コード「Sunset」で入手できる「フュージョンクリスタル」を使用して不足分を補い、ユニットを次の星レベルに進化させます。

| アイテムの種類 | 入手先 | 更新間隔 / 制限 |
| :--- | :--- | :--- |
| **召喚チケット** | アイテムショップ | 5分ごと |
| **フュージョンクリスタル** | アイテムショップ / コード | レア度により異なる |
| **餅（食べ物）** | 放置道場 / ショップ | 受動的に生成 |
| **ミシックシャード** | PvP報酬 | 毎日（上位100プレイヤー） |

## 高度な戦略：サンセットシティとその先へ

サンセットシティのアップデートでは難易度が大幅に上昇し、プレイヤーは新しいレベル上限である120に到達する必要があります。このマップは5つの異なるステージで構成され、最終的にはサイバーDJニッキとの戦いで最高潮に達します。この遭遇を生き抜くには、ユニット図鑑（Unit Index）UIを活用して、収集したヒーローからジェム報酬を受け取り、それをレジェンダリーバナーに再投資する必要があります。

### アビリティアップグレードのためのリソース稼ぎ
**サモンヒーローズのヒーローアビリティ**を最高のパフォーマンスに保つには、毎日のリソース収集ループに参加する必要があります。
*   **ナイトメア・サーカス：** このエンドレスウェーブモードは、ジェムの最高の供給源です。レベル130に到達すれば、わずか数分で300ジェム以上を獲得できます。1日3回まで入場可能です。
*   **放置道場（AFK Dojo）：** プレイしていないときは、キャラクターを道場に置いておきましょう。ここでは1時間あたり約500ジェムが生成されるほか、ユニットのレベルアップに使用する「食べ物」アイテム（餅など）も安定して手に入ります。
*   **PvP報酬：** 上位100人のプレイヤーに食い込むことができれば、ミシックシャードを受け取ることができます。これはゲーム内で最も希少なリソースであり、排出率0.1%のミシックユニットを製作するために必要です。

## FAQ

**Q: エンドゲームに向けて最強のサモンヒーローズのヒーローアビリティを解放するにはどうすればよいですか？**

A: レベル上限120への到達と、サンセットシティ・マップの周回に集中してください。ボスDPS用のフレームレートや、範囲攻撃用のテクノマスターなどのレジェンダリーユニットを優先すべきです。放置道場を活用して、メインチームをレベル上限に保つための十分な食べ物を集めましょう。

**Q: 合成には異なるユニットを使用できますか？それとも同じキャラクターである必要がありますか？**

A: 星のレア度が同じであれば、どのユニットでも使用できます。例えば、星4のユニットであれば、星4のドラグーンに合成して星5にアップグレードすることができます。これにより、「ハズレ」の召喚も進行において非常に価値のあるものになります。

**Q: 2026年において最も速くジェムを入手する方法は何ですか？**

A: 能動的な方法では「ナイトメア・サーカス」が最も効率的で、数分のプレイで数百ジェムを獲得できます。受動的な収入としては「放置道場」が最適で、PCやコンソールから離れている間も1時間あたり約500ジェムを得られます。

**Q: ミシックユニットに天井システムはありますか？**

A: いいえ、天井システムはレジェンダリーユニット（100回召喚で確定）にのみ適用されます。ミシックユニットのドロップ率は0.1%のままですが、PvPで上位にランクインしてミシックシャードを獲得することで、最終的に製作することが可能です。
