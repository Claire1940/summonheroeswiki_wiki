## Troubleshooting Code Issues

It can be frustrating when a code returns an "Invalid" message. Before giving up, check the following common issues that players encounter when trying to use **summon heroes codes**:

1.  **Leading or Trailing Spaces**: When copying and pasting, it is easy to accidentally include a space at the beginning or end. This will cause the system to reject the code.
2.  **Expiry Dates**: Most codes are time-limited. Developers often disable codes from previous months without notice.
3.  **Server Lag**: If the game servers are under heavy load (especially after a major update), the redemption system might take a few seconds to process. Do not spam the button; wait for a confirmation message.
4.  **Version Mismatch**: Ensure you are playing on the latest version of the game. If your client is outdated, newer codes may not be recognized by the server.
